# Array and Object

## Object

| Method         | Big O |
| -------------- | ----- |
| Insertion      | O (1) |
| Removal        | O (1) |
| Searching      | O (N) |
| Access         | O (1) |
| Object.keys    | O (N) |
| Object.values  | O (N) |
| Object.entries | O (N) |
| hasOwnProperty | O (1) |

### Traits

- Key-value pairs
- Fast access, insertion and removal

## Array

| Method    | Big O               |
| --------- | ------------------- |
| Insertion | O (N) or Push O (1) |
| Removal   | O (N) or Pop O (1)  |
| Searching | O (N)               |
| Access    | O (1)               |
| Sort      | O (N * logN)        |

### Traits

- When you need order
